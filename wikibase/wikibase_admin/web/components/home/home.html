Wiki Administrator

The Source tab is where you can manage data in the Wikibase. You can inject new data or update existing data to the Wikibase.
The Property tab is where you can view and manage the properties of the items in the Wikibase. You can add new properties, edit existing ones, and more.
The Settings tab is where you can customize the behavior of the app. You can change various settings such as the language, theme, and more.
